
# Frontend con Vanilla

Proyecto aplicacion desarrollada con Vanilla y Vite.

## Descripción

Esta aplicación consume datos desde la API pública JSONPlaceholder utilizando fetch().


## Estructura del proyecto

```txt
src/
│
├── services/
│   └── userService.js
│
├── menu.js
├── main.js
├── usuarios.js
├── posts.js
├── todos.js
├── about.js
└── style.css
```
